
HTML TAGS:
==========
HEADING:-
---------------
SYNTAX:
	<h1 [align="left"|"center"|"right"]>.................</h1>
EXAMPLE:
<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
      <H1 > HELLO WORLD </H1>
      <H2 > HELLO WORLD </H2>
      <H3 > HELLO WORLD </H3>
      <H4 > HELLO WORLD </H4>
      <H5 > HELLO WORLD </H5>
      <H6 > HELLO WORLD </H6>
  </body>
</html>

PARAGRAPH:-
--------------------
SYNTAX :
	<P [align="left"|"center"|"right"]> .............</p>
EXAMPLE:
<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
      <P> Hello all of you</P>
  </body>
</html>

HORIZONTAL  LINES:
--------------------------------
SYNTAX :
	<hr [align="left"|"center"|"right"] [size="n"] [noshade] [width="nn%">
EXAMPLE:
<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
      <P> Hello all of you</P>
      <hr align="left" width = "30%" noshade>
  </body>
</html>

SUB,SUP,B,I:
-----------------
EXAMPLE:
<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
      <H3> SOME TAGS </H3>
      <hr align="left" width = "40%" noshade>
      <p> here are some text using sub,sup,b,i</p>
      <h4 > BOLD </h4>
      <p><b>Bold</b></p></p>
      <h4 > SUB </h4>
      <P>3<sub>4</sub></P>
      <h4 > SUP </h4>
      <P>3<sup>4</sup></P>
      <h4 > ITALIC </h4>
      <P>HELLO <i>world</i></P>
  </body>
</html>

PRE:
------
	It will display with the exact lines entered by the user in the code.
SYNTAX:
	<pre>...........</pre>
EXAMPLE:<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
      <H3> PRE TAG </H3>
      <pre>
        Hello
        World
      </pre>
  </body>
</html>

&AMP,&LT,&GT,&COPY:
----------------------------------
EXAMPLE:
<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
      <p>hello &amp; world</p>
      <br>
      <p> 2 &lt 3</p>
      <br>
      <p> 3 &gt 2</p>
      <br>
      <p>&quot; Hello World &quot;</p>
      <br>
      <p>&copy; hello</p>
  </body>
</html>
