SELCTORS IN CSS:
==============
	A CSS selector selects the HTML element(s) you want to style..
	1.simple
	2.id
	3.class
	4.universal
	5.grouping
1.SIMPLE SELECTOR:
	The element selector selects HTML elements based on the element name.
EXAMPLE:
<!DOCTYPE html>
<html>
<head>
<style>
body{
	background-color:green;
    }
p {
  text-align: center;
  font-size:200%;
  color: red;
} 
</style>
</head>
<body>

<p>Every paragraph will be affected by the style.</p>
<p id="para1">Me too!</p>
<p>And me!</p>

</body>
</html>


2.ID SELECTOR:
	ID  Selector is used to select specific element.
	"#" is used while declaring.
EXAMPLE:
<!DOCTYPE html>
<html>
<head>
<style>
#para1 {
  text-align: center;
  font-size : 200%;
  color: red;
  
}
#bd{
	background-color:green;
    }
</style>
</head>
<body id = "bd">

<p id="para1">Hello World!</p>
<p>This paragraph is not affected by the style.</p>

</body>
</html>


3.CLASS SELECTOR:
	CLASS Selector is used to select specific class attribute.
	"." is used while declaring.
EXAMPLE:
<!DOCTYPE html>
<html>
<head>
<style>
.bd{
	background-color:green;
    }
p.center {
  text-align: center;
  color: red;
.center {
  text-align: center;
  color: red;
p.large {
  font-size: 300%;
}
</style>
</head>
<body class = "bd">

<h1 class="center">Red and center-aligned heading</h1>
<p class="center">This paragraph will be red and center-aligned.</p> 
<p class="center large">This paragraph will be red, center-aligned, and in a large font-size.</p> 
</body>
</html>

4.UNIVERSAL SELECTOR:
	Selects all the html slements on the page.
	Uses "*" while declaring.
EXAMPLE:
<!DOCTYPE html>
<html>
<head>
<style>
* {
  text-align: center;
  color: blue;
}
.gg{
	color:green;
    }
</style>
</head>
<body>

<h1>Hello world!</h1>

<p>Every element on the page will be affected by the style.</p>
<p id="para1">Me too!</p>
<p class = "gg">And me!</p>

</body>
</html>

5.GROUPING SELCTORS:
	Selects all the html elements with the same style deefenitions.
EXAMPLE:
<!DOCTYPE html>
<html>
<head>
<style>
h1, h2, p {
  text-align: center;
  margin-left:40px;
  color: red;
}
</style>
</head>
<body>

<h1>Hello World!</h1>
<h2>Smaller heading!</h2>
<p>This is a paragraph.</p>

</body>
</html>
