BITWISE OPERATORS IN JAVA SCRIPT
==============================
&	-AND
|	-OR
^	-XOR
~	-NOT
<<	-ZERO FILL LEFT SHIFT
>>	-SIGNED RIGHT SHIFT
>>>	-ZERO FILL RIGHT SHIFT

AND(&):
-------------
<!DOCTYPE html>
<html>
<body>

<h2>JavaScript Bitwise AND</h2>

<p id="demo">My First Paragraph.</p>

<script>
document.getElementById("demo").innerHTML = 5 & 1;
</script>

</body>
</html>
